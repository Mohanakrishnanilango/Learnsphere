# LearnSphere

LearnSphere is an e-learning platform designed to provide students with an interactive and accessible learning experience. The platform enables users to explore courses, track their learning progress, and access educational resources in one place.

## Features

* User authentication and secure login
* Browse and enroll in courses
* Interactive learning modules
* Progress tracking
* Responsive and user-friendly interface
